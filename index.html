Comfy Homepage é um projeto open-source desenvolvido por @dxppwr, oferecendo uma experiência de navegação moderna e personalizável.

O código-fonte está disponível em: github.com/dxppwr/home

Privacidade: Este projeto não salva nenhuma informação pessoal inserida pelo usuário. Todos os dados são armazenados localmente no seu navegador e podem ser removidos a qualquer momento.

Desenvolvido com tecnologias web modernas:

  • HTML5 para estrutura semântica
  • CSS3 com variáveis personalizadas e animações suaves
  • JavaScript puro (Vanilla JS) para interatividade
  • APIs RESTful para integração com serviços externos
  • LocalStorage para persistência de dados

Recursos e personalizações disponíveis:

  • Múltiplos motores de busca (Google, DuckDuckGo, Brave, Kagi, etc.)
  • Pesquisa por imagem com suporte a upload e colagem
  • Previsão do tempo em tempo real
  • Links personalizáveis com gerenciamento intuitivo
  • Tema claro/escuro com detecção automática
  • Suporte a múltiplos idiomas (Português, Inglês, Espanhol)
  • Interface responsiva e adaptável

Este projeto é mantido com carinho e dedicação. Se você gostou e quer apoiar o desenvolvimento, doações são muito bem-vindas!
